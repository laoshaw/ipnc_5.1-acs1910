Codesourcery code generation targets for Arm processors

  • XDCspec declarations
This package supports two Codesourcery gcc targets. One is GCArmv5T for code compatible with Arm v5T architecture, and another one is GCArmv6 for Arm v6 architecture. These targets are deprecated, and the targets from gnu.targets.arm should be used instead.
