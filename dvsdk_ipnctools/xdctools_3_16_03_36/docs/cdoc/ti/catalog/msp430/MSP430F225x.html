MSP430F225x Cpu definition

  • XDCscript usage
  • XDCspec declarations
  • Individual elements
This device has 16KB + 256B of Flash Memory and 512B RAM. The MSP430F2254 has two op-amps (OA) whereas the MSP430F2252 has none.
